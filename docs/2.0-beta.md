# elm-pages 2.0 beta

## Setup Instructions

```shell
npx dillonkearns/elm-pages#static-files init elm-pages-demo
cd elm-pages-demo
git init
git submodule add -b static-files https://github.com/dillonkearns/elm-pages.git
npm install
npm start
```
